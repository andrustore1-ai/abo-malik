# إدارة المحروقات v10 النهائي

هذه النسخة تستخدم Realtime Database REST بنفس طريقة المشروع المرفق الشغال، ولا تعتمد على Firestore.

## التشغيل
1. ارفع كل الملفات مباشرة بدون مجلد رئيسي.
2. افتح Firebase Console.
3. ادخل إلى Realtime Database > Rules.
4. انسخ محتوى ملف REALTIME_DATABASE_RULES.txt واضغط Publish.
5. افتح الموقع من رابط HTTPS.
6. امسح كاش المتصفح أو بيانات الموقع عند استبدال نسخة قديمة.
